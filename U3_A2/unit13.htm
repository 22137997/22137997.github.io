Introduction:
Unit 13 is website development, where we anaylse, design and create our own website. In assignment one, we were tasked to assess the uses and features of websites, we were chosen between four, Switch, Gocompare, Mobiles and Mobliesphonedirect. I chose Gocompare and Mobilephonesdirect, identifing their website components, such as; templates, online forums, hot spots, action buttons etc. then near the end of the assignment we had to identify their improvised features, like presentation, usabiliity, accessibility and performance. Then to compleet teh assigmnet, we listed the strengths and weaknessess of both websites.

In assignment two, we were tasked to design a website dedicated to advertising a varienty of mobile phones. I first started out designing prototypes, as well what I was going to include. I then created a structure chart, showing the positions of the pages and whom they are connected to, which was followed up by my first design for the hompage. Each page consisted of a list of assets that were going to be included, like images, videos, links and tables. Once all my pages were blueprinted, I filled in the assets log, which would show what the images of the website would look like as well as their name and role on the pages, after that I filled in the test log, listing down what I was going to test when the website was completed.

In assigment three came to developing the website, creating this wasn't as hard as thought, most of the code was pretty simple to understand and apply too. I created each page via HTML code, including the images, videos, table, links etc. With the assets folder containg most of the vital components. Then came testing the website, a few things wouldn't load in correctly, such as the video or certain moving images, but eventually I managed to fix the bugs in the code and made them fully working.

To finish off woth assignment four, I obtained feedback from others to see how well my website performed and if it met the criteria, all of the feedback was very positive, with me only needing to make small changes here and there. Then when the website was fully complete, I uploaded it to Github, finishing off Unit 13.
